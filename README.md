# MyDay Planner

Simple Flutter app for a CIS final project.

Features:
- Well-designed UI with cards and list tiles
- Local data handling using Hive
- Web-based weather service (Open-Meteo)
- Parsing a local JSON file for motivational tips
- Proper division into widgets, screens, and services
- Popup dialog to add tasks, menu in the app bar, and a floating action button
